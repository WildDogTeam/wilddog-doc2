title: LocalStreamOptions.Builder
-------------------------

设置多媒体采集参数的对象。

## 方法

### captureVideo(boolean)

**定义**   

```java
Builder captureVideo(boolean enable)
```

**说明**

设置本地视频流的视频开关,默认为开。

**参数**

| 参数名 | 描述 |
|---|---|
|enable|本地视频流的视频开关。|

**返回值**

当前媒体采集参数构建对象。
