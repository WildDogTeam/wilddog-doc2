## 方法

### credential

**定义**

```js
credential(email, password)
```

**参数**

| 参数名      | 说明             |
| -------- | -------------- |
| email    | string 类型，邮箱地址 |
| password | string 类型，密码信息 |

**返回值**

[wilddog.auth.Credential](/api/auth/web/Credential.html)
