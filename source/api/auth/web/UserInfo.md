
title: UserInfo
---

UserInfo 对象包含所有维护用户登录方式个人信息的接口，我们不能直接创建此对象。

## 属性

### displayName

**定义**

```js
nullable string
```

**说明**

用户名称

</br>

------

### email

**定义**

```js
nullable string
```

**说明**

用户主邮箱地址

<!-- </br>

### phone

**定义**

```js
nullable string
```

------

### photoURL

**定义**

```js
nullable string
```

**说明**

用户照片地址

</br>

### providerId

**定义**

```js
string
```

**说明**

当前用户登录使用的身份认证提供商名称，例如 weibo，weixin。

</br>

------

string
